Obsidian 跨设备同步

Posted on 2024-04-18 In 常用工具
Symbols count in article: 755 Reading time ≈ 1 mins.

概述

Obsidian 跨设备数据同步主要有下面几个思路：

  1. 购买Obsidian官方服务
  2. 将仓库存放在 IOS 的 iCloud 中，缺点是 Windows 系统不能同步使用
  3. 坚果云
  4. Github + obsidian-git 插件
  5. 使用阿里云OSS服务与Obsidian中的Remotely Save插件
    官方服务价格高昂，阿里云OSS服务40G-5年-45RMB，非常实惠

思路五-阿里云 OSS 和 Remotely Save 插件

适合同时有 Mac、IOS、Windows 使用需求的用户，在不同系统中实现同步只需要下载Obsidian软件后，重复执行配置Obsidian这个步骤即可

阿里云对象存储 OSS 资源包

Bucket服务配置

无特殊要求，使用默认设置创建即可
